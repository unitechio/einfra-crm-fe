# Admin Dashboard

This is a comprehensive admin dashboard built with Next.js, TypeScript, and Tailwind CSS. It provides a modern and intuitive interface for managing servers, users, and other aspects of your application.

## Live Demo

A live demo of the application can be found here: [https://your-live-demo-url.com](https://your-live-demo-url.com)

## Features

*   **Authentication:**
    *   Login and Registration: Secure user authentication and registration.
    *   Session Management: Keep users logged in with session management.
*   **Server Management:**
    *   Server List: View a list of all servers with their status.
    *   Server Details: View detailed information about each server.
    *   Server Actions: Start, stop, and restart servers.
*   **User Management:**
    *   User List: View a list of all users with their roles and status.
    *   User Details: View detailed information about each user.
    *   User Actions: Create, edit, and delete users.
*   **Role-Based Access Control (RBAC):**
    *   Permissions: Assign permissions to roles to control access to resources.
    *   Roles: Create and manage roles to group users with similar permissions.
*   **Real-time System Monitoring:**
    *   Dashboard: A real-time dashboard with charts and graphs to monitor system health.
    *   Alerts: Receive alerts when system resources reach critical levels.
*   **Audit Trail:**
    *   Log Viewer: View a log of all user activity within the application.
*   **Notifications:**
    *   Real-time Notifications: Receive real-time notifications for important events.

## Styling and Theming

*   **Dark and Light Modes:** Switch between dark and light modes for a comfortable viewing experience.
*   **Customizable Themes:** Easily customize the look and feel of the application to match your brand.
*   **Responsive Design:** The dashboard is fully responsive and works on all screen sizes.

## Built With

*   **[TypeScript](https://www.typescriptlang.org/):** A typed superset of JavaScript that compiles to plain JavaScript.
*   **[Tailwind CSS](https://tailwindcss.com/):** A utility-first CSS framework for rapid UI development.
*   **[Shadcn/ui](https://ui.shadcn.com/):** A collection of re-usable components built with Radix UI and Tailwind CSS.
*   **[Zustand](https://github.com/pmndrs/zustand):** A small, fast, and scalable state management solution for React.
*   **[Axios](https://axios-http.com/):** A promise-based HTTP client for the browser and Node.js.
*   **[Socket.io-client](https://socket.io/docs/v4/client-api/):** A real-time engine for web applications.

## Project Structure

The project is structured as follows:

*   **`src/app`:** Contains the main application code, including pages, layouts, and components.
*   **`src/components`:** Contains re-usable UI components.
*   **`src/lib`:** Contains utility functions and libraries.
*   **`src/services`:** Contains services for interacting with external APIs.
*   **`src/store`:** Contains the application's state management logic.
*   **`public`:** Contains static assets, such as images and fonts.
